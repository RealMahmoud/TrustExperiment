How does it work ?

Have a fully set-up and synced node. Use the Idena sign in process to prove your node is setup. The person sharing the invite can now check your address to make sure it is properly set up to receive an invite

Can I get an invite?

sure!

1. Have a fully synced and set up Idena Node

2. Use the Idena Sign in

3. Your node is now checked to be set up. Join our Telegram and we will verify you're a human and good actor for Idena Network

4. Pass the Idena Validation ceremony and become a node for the Idena Network
